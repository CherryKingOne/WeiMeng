📋 核心流程说明

  • 入口: 所有用户消息首先到达 Emma 监听节点
  • 决策: Emma 决策需要哪个专业Agent
  • 激活: 选中的 Agent 从 Dormant 状态转为 Engaged 状态
  • 交互: 用户与专业Agent直接对话，Emma 仅旁听不干预
  • 流转: Agent 可请求流转到其他Agent协作
  • 释放: 任务完成后 Agent 回到 Dormant 状态
  • 响应: Emma 汇总结果回复用户
